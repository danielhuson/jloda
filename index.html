Welcome to the JLODA project

JLODA is a Java library of data structures and algorithms. This library contains a number of packages that are shared by a number of Java programs. such as SplitsTree, Dendroscope and MEGAN.

Graphs

JLODA provides a graph package that is very loosely based on the graph class provided by the LEDA C++ library

GUI support

There are a number of supporting classes for the GUIs provided by programs.

Authors and Contributors

Most of the code was written by Daniel Huson (@danielhuson) as part of his research as Professor of Algorithms in Bioinformatics at the University of Tuebingen, starting 2002.

Programs that are based on JLODA

  • MEGAN
  • Dendroscope
  • SplitsTree4
  • MALT
  • MetaSim
  • CGViz
